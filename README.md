# Hindi / Marathi Name Finder

A Python tool to extract and search Hindi/Marathi names from voter-list PDFs using OCR.

## Features
- Extracts names from PDFs
- Supports OCR for scanned files
- Transliteration (Hindi → English)
- Fast caching system

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python hindi_name_finder.py /path/to/folder
python hindi_name_finder.py /path/to/file.pdf
```

## Requirements
- Python 3.x
- Tesseract OCR installed
